# Husband Birthday Surprise Website

## Run
```bash
pip install -r requirements.txt
python app.py
```

Open:
```bash
http://127.0.0.1:5000
```

## Add your media
- Opening slideshow photos: `static/images/hero/`
- Timeline photos: `static/images/timeline/`
- Extra photos: `static/images/gallery/`
- Final photo: `static/images/final/`
- Journey video: `static/video/our_journey.mp4`
- Background music: `static/audio/bg_music.mp3`

The app auto-detects images, video, and music from these folders.
